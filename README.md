# mynewvue3.0
* Vue-cli 3.0 初体验

### new vue3.0 aplha
```
npm install -g vue@cli
```
* now create your first new project
```
vue create <your project> 
```

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
